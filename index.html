THE WHO

I'm a self-taught web developer and graphic designer, with a degree in audiovisual communications.

I have experience in front-end and back-end development. Fluent in HTML5, CSS3, ES6, PHP, MySQL. I also have some experience in Node and Electron, and I'm currently learning React.

I've worked on web development almost since the Internet first appeared; back when there was no CSS and JavaScript was just a way to do fancy tricks on a site (Remember the <blink> tag?). By then, the way to really add interactivity and animation to a website was using Flash, so I worked with Flash/ActionScript for a long time.

From BASIC (yes, I know), to Pascal, C, ActionScript, C#, PHP, SQL to JavaScript, coding became a passion. I love the possibility of beating the machine, of solving problems. That road led me to participate in projects for Etheron (the first Venezuela's ISP), Kellogg's, Mitsubishi, Jones Knowles Ritchie, PageUp; and develop interactive pieces for the likes of world-renowned plastic artists Carlos Cruz-Diez and Rolando Peña.

That contact with the art and design world taught me a lot, so I've mastered tools like Photoshop, InDesign, Illustrator, Blender and many others; and had the opportunity to teach in several academies and institutes.

BITS OF ME

SyncLoop

Developed in C#, SyncLoop is an application I created to help with the synchronization of video loops for dubbing and subtitling.

The app allows the user to mark the in and out points of each character intervention, and generates a final file according to the format specified by the client.

It also lets the user input program info, create, edit and delete characters, find and replace text, offset loops, check for correct loop order, determine the total lines for each characters (as needed to calculate the actors' pay), keep track of current rates and calculate the period earning for the translators.

VideoCat

Developed in Electron (JavaScript, Node), VideoCat is a video cataloger that allows the user to classify and tag videos. It uses VirtualDub, a free video editor, to automaticcaly generate thumbnails from de videos. It also allows for searchs based on video cast, tags and scores.
